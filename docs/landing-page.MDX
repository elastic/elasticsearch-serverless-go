This is the official Go client for the **Elasticsearch Serverless** service. 

If you want to develop your Go application with the Elasticsearch Stack, refer 
to the [Elasticsearch Client](https://github.com/elastic/elasticsearch-go) 
instead. 

If you want to develop your Go application with Elastic Enterprise Search, 
refer to the 
[Enterprise Search Client](https://github.com/elastic/enterprise-search-go/).
